# Sistema de Gestión de Inventario - BroadSonick

Proyecto desarrollado en JavaScript utilizando:

- Arreglos
- Objetos
- Ciclos for y while
- Funciones
- Validaciones con if/else
- Métodos String
- Objeto Math

Permite:

- Agregar productos
- Eliminar productos
- Calcular valor total del inventario
- Aplicar impuesto automático

Desarrollado como práctica del módulo M3 - Clase 9.
